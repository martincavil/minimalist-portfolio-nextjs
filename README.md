# Minimalist Portfolio (Next.js)

Welcome to my minimalist portfolio project, built with Next.js, React, and Tailwind CSS. This website serves as my personal portfolio, showcasing my work, skills, and background as a freelance developer.

## Features

- Clean, minimalist design
- Responsive layout
- Multilingual support (English & French)
- Projects, skills, and blog posts sections
- SEO optimized with Open Graph and Twitter meta tags

## Purpose

This project is designed to be a fast, modern, and minimal portfolio site, focusing on clarity and simplicity. It is fully open-source and can be used as inspiration or a starting point for your own portfolio.

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
